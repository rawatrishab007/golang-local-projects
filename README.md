Practice Projects: Go Backend & CLI Tools
This repository is a collection of personal projects developed to practice and master Golang, focusing on CLI tool architecture, file I/O, and concurrency patterns.

🛠 Tech Stack
Language: Go (Golang)

Concepts: Concurrency (Goroutines/Channels), File Handling, JSON Processing, CLI Flags.

🚀 Included Projects
1. Days Calculator
A utility tool designed to calculate the precise number of days between dates or convert years into days, handling leap years and date formatting.

Key Learning: Date-time manipulation in Go's time package.

2. Student Grade Manager
A management system that allows users to input student data, calculate averages, and assign grades. It supports data persistence via JSON/CSV files.

Key Learning: Structs, interfaces, and file serialization.

3. URL Reporter
A tool that takes a list of URLs and reports their current HTTP status codes, helping identify broken links or server downtime.

Key Learning: Network programming with net/http and error handling.

4. Concurrent File Searcher
A high-performance CLI tool that searches for specific patterns or filenames within a directory tree using Go's concurrency primitives.

Key Learning: Scaling tasks with Goroutines and synchronizing data with WaitGroups and Channels.
